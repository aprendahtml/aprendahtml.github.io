Tabelas

As tabelas, no html, são definidas pelo elemento table, mas esse elemento sozinho não produz nenhum efeito no navegador, ele precisa de outros elementos para mostrar a tabela no navegador, vamos ver agora esses outros elementos para montar uma tabela de forma semântica no html.

Primeiro devemos utilizar o elemento table, que define uma tabela.

Podemos, ou não, utilizar o elemento caption, para definir uma legenda ou um título para a tabela.

Podemos, ou não, utilizar os elementos semânticos thead para definir o cabeçalho da tabela, tbody para definir o corpo da tabela e tfoot para definir o rodapé da tabela, não importa a ordem em que esses elementos são escritos, o thead sempre mostrará os itens na parte de cima da tabela, o tfoot sempre mostrará os itens na parte de baixo da tabela e o tbody sempre mostrará os itens entre o thead e o tfoot.

Devemos utilizar o elemento tr (table row), que define uma linha na tabela. O tr também não produz nenhum efeito no navegador.

No thead, dentro do tr, utilizamos o th (table headings), que define o título do cabaçalho da tabela.

No tbody e no tfoot, dentro do tr, utilizamos o td (table data), que define o dado da tabela. A diferença do th para o td, além da semântica, é que o th já aparece formatado em negrito e centralizado.

Para definir as linhas numa tabela, utilizamos o elemento tr, não temos um elemento para definir as colunas, as colunas são definidas pelo número de td dentro de cada tr ou th, por exemplo, se tivermos 4 td dentro da tr então teremos 4 colunas na tabela.

Para unir 2 colunas numa mesma linha, utilizamos o atributo colspan no elemento td, o valor do colspan é o número de colunas que queremos unir, no exemplo abaixo utilizamos o valor 2. Note que nesse exemplo, temos 4 colunas e como vamos unir 2 delas, o número de td dentro dessa tr, passa a ser 3 td, pois a 2ª td foi unida com a 1ª.

Para unir 2 linhas numa mesma coluna, utilizamos o atributo rowspan no elemento td, o valor do rowspan é o número de linhas que queremos unir, no exemplo abaixo utilizamos o valor 2. Note que nesse exemplo, temos 6 linhas e a diferença para o colspan é que não diminuímos o número de td dentro dessa tr, nós devemos diminuir o número de td na tr de baixo, nessa tabela, cada tr tem 4 td, mas a tr abaixo do rowspan deve ter 3 td, pois uma vai se unir com o rowspan da tr acima.

No arquivo css, além dos estilos já vistos, apliquei uma borda, com a propriedade border, e com o valor 2px solid #000, nos seletores table, th e td, esse estilo criou uma borda para cada elemento, para juntar essas bordas, utilizei no seletor table a propriedade border-collapse com o valor collapse.

Código html:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <link rel="stylesheet" href="exemplo-tabelas.css">
    <meta charset="utf-8">
    <title>Exemplo Tabelas</title>
</head>
<body>
    <table>
        <caption>Legenda ou Título da Tabela</caption>
        <thead>
            <tr>
                <th>Mês</th>
                <th>Entrada</th>
                <th>Saída</th>
                <th>Saldo</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Janeiro</td>
                <td>R$ 1.000,00</td>
                <td>R$ 500,00</td>
                <td>R$ 500,00</td>
            </tr>
            <tr>
                <td>Fevereiro</td>
                <td>R$ 800,00</td>
                <td>R$ 500,00</td>
                <td>R$ 300,00</td>
            </tr>
            <tr>
                <td>Março</td>
                <td>R$ 1.500,00</td>
                <td>R$ 800,00</td>
                <td>R$ 700,00</td>
            </tr>
            <tr>
                <td colspan="2">Colspan com 2 colunas</td>
                <td>Teste</td>
                <td>Teste</td>
            </tr>
            <tr>
                <td rowspan="2">Rowspan com 2 linhas</td>
                <td>Teste</td>
                <td>Teste</td>
                <td>Teste</td>
            </tr>
            <tr>
                <td>Teste</td>
                <td>Teste</td>
                <td>Teste</td>
            </tr>
        </tbody>
        <tfoot>
            <tr>
                <td colspan="4">Saldo Total: R$ 1.500,00</td>
            </tr>
        </tfoot>
    </table>
</body>
</html>

Código css:

table {
    width: 500px;
    padding: 5px;
    border: 2px solid #000;
    border-collapse: collapse;
}

thead {
    font-family: Arial, sans-serif;
    background-color: #777;
    color: #fff;
}

tbody {
    background-color: #efefef;
}

tfoot {
    font-family: Arial, sans-serif;
    background-color: #777;
    color: #fff;
    text-align: center;
    font-weight: bold;
}

th {
    border: 2px solid #000;
    padding: 5px;
}

td {
    border: 2px solid #000;
    padding: 5px;
}

caption {
    font-family: Arial, sans-serif;
    font-weight: bold;
    margin-bottom: 5px;
}
