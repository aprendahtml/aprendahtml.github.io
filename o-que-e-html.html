O que é HTML?

HTML é a abreviação de Hyper Text Markup Language, ou Linguagem de Marcação de Hipertexto, e é a linguagem utilizada para a exibição de páginas na Internet.

O html é uma linguagem de texto puro e a sua sintaxe básica é feita através de tags e elementos html. As tags são os parênteses angulares, que são os caracteres de menor que (<) e maior que (>) e as tags de abertura e de fechamento têm um elemento html dentro dele para que o navegador possa ler e executar esse elemento, por exemplo, o elemento body é escrito dessa forma:

<body>

O exemplo acima é a abertura da tag com o elemento body, o fechamento de uma tag é escrita conforme a abertura, com a diferença que antes do nome da tag tem o caracter barra (/), por exemplo o fechamento da tag com o elemento body é escrita dessa forma:

</body>

As tags de elementos que são abertas e fechadas são chamadas de normal elements e as tags de elementos que não são fechadas são chamadas de void elements.

As tags de elementos que não são fechadas, as void elements, são area, base, br, col, embed, hr, img, input, keygen, link, meta, param, source, track e wbr.

Existem elementos que podem ter atributos e elementos que obrigatoriamente têm que ter atributos. Os atributos são sempre passados depois do elemento na tag de abertura.

Exemplo de elemento que pode ter atributo:

<html lang=”pt-br”></html>

Exemplo de elemento que tem que ter atributo:

<img src=”foto.jpg”>
