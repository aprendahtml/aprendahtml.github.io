nav e Listas

Como dito anteriormente, devemos trocar a div que envolve o Menu de Navegação pelo elemento semântico nav, que representa exatamente um menu de navegação para as páginas do site. Não se deve usar o nav para todos os conjuntos de links do site, por exemplo, não se usa o nav em links de paginação e em links para outras páginas, como links para Mídias Sociais. A troca de div para nav não altera nada visualmente mas altera a semântica do site para os motores de busca da Internet.

Normalmente, na Internet, para Menu de Navegação, se usa uma lista com links para acessar as páginas do site, como exemplo, vamos criar links para o Home, Fotos, Vídeos e Contato. Nesse caso essa lista não precisa ser ordenada, nem por número nem por letra, então vamos utilizar uma lista não ordenada com o elemento html ul (unordered list), dentro da tag ul, utilizamos o elemento html li (list item) para escrever o nome dos itens da lista.

Note que os nomes da lista ficaram um abaixo do outro, mas nós queremos que fique um ao lado do outro e por padrão o elemento ul imprime os bullets, aquelas bolinhas ao lado do nome, que também não queremos. Para retirar os bullets, criamos um class no ul com o nome menu, e no arquivo css criamos para esse class a propriedade list-style com o valor none.

Para que os itens da lista fiquem um ao lado do outro, adicionamos no arquivo css, no li que pertence ao class menu, a propriedade display com o valor inline-block.

As mudanças efetuadas fizeram com o nav (onde está a lista e os itens da lista) ficasse maior que o header (a faixa com o fundo amarelo), descentralizando o h1 (o título Receitas de Bolo), para resolver esse problema, adicionei no arquivo css, no class nav, a propriedade height com valor de 80 pixels, que é a altura do header e ainda adicionei a propriedade box-sizing com o valor border-box, para que ele mantenha esse valor, mesmo se for adicionado padding e border.

Vamos separar um pouco os nomes dos itens da lista e separar o último nome da lista da lateral do header, acrescentando no li do class menu, a propriedade margin-right com o valor de 20 pixels.

Ainda no .menu li, vamos mudar a font com as seguintes propriedades e valores:
font-family: "Arial", sans-serif;
font-weight: bold;
text-transform: uppercase;
font-size: 20px;

Agora vamos criar os links clicáveis para a lista de itens da lista não ordenada. Para isso usamos no arquivo html, o elemento html a, de anchor ou âncora, com o atributo obrigatório href, o valor do href é o caminho para o link.

Note que o elemento a incluiu automaticamente a cor azul no link e o underline. Podemos modificar essas propriedades no arquivo css, no seletor .menu li a, adicionando a propriedade text-decoration com o valor none, para retirar o underline e adicionando a propriedade color com o valor #000 em hexadecimal para definir a cor preta. Repare que ao passar o mouse sobre os links o cursor muda, indicando ser um link, e se for clicado será redirecionado para uma página inválida porque ainda não temos as páginas indicadas pelo href.

Vamos utilizar no arquivo css, o pseudo seletor hover, para quando o mouse passar por cima do link, mudar uma ou mais propriedades css, nesse exemplo, vamos mudar a cor da fonte de preto para cinza. O hover é escrito com dois pontos (:) depois do nome do elemento, nesse exemplo é escrito assim: .menu li a:hover, adicionamos a propriedade color com o valor #686868 em hexadecimal.
