Iniciando com CSS

A partir de agora, que vamos iniciar com o CSS e que vamos ter que fazer testes de layout, é muito importante que você mantenha aberto os arquivos html e css no editor de texto, no nosso caso o Sublime Text, e que mantenha aberto o arquivo html no navegador, no nosso caso o Google Chrome, para que você possa ver o resultado da renderização no navegador de cada teste de tags html e de atributos css.

Foi mostrado, ainda, muito pouco de HTML, mas como o CSS é muito atrelado ao estudo do html, resolvi ir mesclando os assuntos, até mesmo porque para testar e demonstrar alguns aspectos do html vamos precisar de alguns atributos do css.

Veremos, então, uma pequena introdução ao CSS:

Cascading Style Sheets, folhas de estilo em cascata é o responsável pela formatação dos estilos do arquivo html, ou seja, é com o CSS que definimos o tipo, o tamanho e a cor da fonte de cada elemento html, a cor de fundo, os espaçamentos internos e externos, as bordas, os alinhamentos, as transições e vários outros atributos de estilo.

O CSS pode ser inline, interno e externo. A recomendação é que se aplique o CSS externo, mas veremos as 3 aplicações.

O CSS inline é aplicado dentro do arquivo html, logo após um elemento html, através do atributo style, por exemplo: 

<body>
    <p style="color: blue; font-family: Arial;">
        Testando o CSS inline.
    </p>
</body>

No exemplo acima, será aplicado a fonte Arial com a cor azul somente a essa tag p.

O CSS interno é aplicado dentro do arquivo html, dentro da tag <head>, utilizando a tag <style>, por exemplo: 

<head>
    <style>
        h2 {font-family: Arial; color: blue; font-size: 14pt;}
    </style>
</head>

Note que o código CSS é aplicado em todos os elementos h2 do arquivo html, por isso o “em cascata” em seu nome, mas com o CSS podemos aplicar estilos diferentes para elementos de mesmo nome, usando os atributos class e id, mas veremos esses atributos mais a frente, no CSS externo.

O CSS externo é o mais recomendado para ser aplicado. Nesse caso, criamos um arquivo com extensão .css, que tem que ser referenciado em todas as páginas html dentro da tag <head>, utilizando a tag <link>, por exemplo: 

<head>
    <link rel="stylesheet" href="css/estilo.css">
</head>

Note que o atributo href indica o caminho onde está o arquivo css que queremos utilizar, nesse caso o arquivo é estilo.css que está dentro da pasta css.

Sintaxe do arquivo css:

seletor {
    propriedade: valor;
}

Nome do seletor, chave aberta ({), nome da propriedade, dois pontos (:), valor para a propriedade, ponto e vírgula (;) e chave fechada (}).

Se o seletor só tiver uma propriedade, não é obrigatório o uso do ponto e vírgula após o valor, e se tiver mais de uma propriedade também não é obrigatório o seu uso após o valor da última propriedade, mas por convenção, se usa o ponto e vírgula, até porque, se adicionarmos outra propriedade ao seletor, ele já estará com o ponto e vírgula, que nesse caso já seria obrigatório.

Crie um arquivo teste-css.html e outro arquivo teste-css.css, por enquanto deixe o arquivo css em branco e no arquivo html escreva o seguinte código: 

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <link rel="stylesheet" href="teste-css.css">
    <meta charset="utf-8">
    <title>Teste CSS</title>
</head>
<body>
    <h1>Teste CSS</h1>
</body>
</html>

Resultado no navegador:

Agora no arquivo css escreva o seguinte código:

h1 {
    color: blue;
    background-color: yellow;
    font-size: 38px;
}

Resultado no navegador:

O seletor é o elemento html h1, as propriedades css dentro das chaves serão aplicadas em todos os elementos h1 dentro do arquivo html.

As propriedades são color, background-color e font-size, que são respectivamente cor da fonte, cor do fundo e tamanho da fonte.

Os valores são blue, yellow e 38px, que são respectivamente azul, amarelo e 38 pixels.
