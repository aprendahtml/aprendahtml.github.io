img / figure / figcaption

Vamos incluir uma imagem no lugar da palavra Logotipo, para isso utilizamos o elemento html img, já gravei a imagem na pasta img no mesmo diretório do arquivo html, isso é muito importante para a organização do diretório e para incluir o caminho correto da imagem no atributo src.

Também vamos incluir uma foto de um bolo de chocolate, e como essa imagem faz parte do conteúdo do site fictício, que é sobre Receitas de Bolo, vamos incluir essa imagem no elemento html semântico figure, e no figure pode ser usado uma legenda para a foto usando o elemento html semântico figcaption, o img e o figcaption devem estar dentro da tag figure.

Na imagem do logotipo do site não foi usado o figure, pois o logotipo não é parte integrante do conteúdo do site.

Para incluir uma imagem, usamos a tag img com o atributo obrigatório src, para indicar o caminho da imagem, usamos ainda o atributo alt, que apesar de não ser obrigatório é muito importante ser usado, até mesmo para os mecanismos de busca da Internet, usamos também o atributo title, que faz aparecer o seu valor quando colocado o mouse sobre a imagem. Poderíamos ainda usar os atributos width e height para definir a largura e a altura da imagem.

O parágrafo Menu de Navegação foi para a linha de baixo, pois o elemento p é um elemento em bloco, mas nós queremos que o Menu de Navegação fique ao lado do logotipo.

Para isso vamos envolver as tag img e p com uma tag div, o Sublime Text tem um atalho de teclado que faz isso automaticamente, selecione o texto que você quer envolver com uma tag de abertura e de fechamento, pressione as teclas Shift + Alt + W e digite o nome da tag que você quer utilizar.

Na div com o logotipo, incluímos o class logo e na div com o Menu de Navegação, incluímos o class nav, mais a frente veremos que podemos usar o elemento semântico nav para defenir um Menu de Navegação do site.

Para deixar o logotipo e o Menu de Navegação na mesma linha, usamos no arquivo css a propriedade display com o valor inline-block para as divs com os class logo e nav.

Faça cada troca no arquivo html e no arquivo css, grave e veja o resultado no navegador, no Sublime Text quando o arquivo não está salvo, aparece ao lado do nome do arquivo um círculo, que indica que o arquivo foi modificado e não foi salvo, e quando o arquivo é salvo, aparece ao lado do nome do arquivo um x, que indica que o arquivo foi salvo e não tem nenhuma modificação a salvar.

Para colocar o Menu de Navegação alinhado à direita no header, usamos no arquivo css, a propriedade float com o valor right.

O logotipo e o Menu de Navegação ficaram muito colados no header, para dar um espaçamento interno no logotipo e no Menu de Navegação, usamos no arquivo css, a propriedade padding com o valor de 10 pixels, tanto no class logo quanto no class nav.

Note que a div do Menu de Navegação ficou um pouco acima do centro do header, para centralizar verticalmente a div com o class nav, vamos utilizar no arquivo css, as propriedades position, top e transform, com os respectivos valores, relative, 50% e translateY(-50%).

Centralizamos os títulos h1 e h2, utilizando no arquivo css, a propriedade text-align com o valor center.

Estilizamos o figcaption, no arquivo css, vamos ver as propriedades usadas:

font-family: "Arial", sans-serif;
Utiliza a fonte Arial, se o usuário não tiver a fonte instalada, utiliza qualquer outra fonte sem serifa (sans-serif).

font-weight: bold;
Aplica um peso na fonte, nesse caso, negrito (bold).

font-style: italic;
Aplica um estilo na fonte, nesse caso, itálico.

background-color: #686868;
Aplica uma cor de fundo, nesse caso, cinza escuro, definido pelo valor em hexadecimal.

color: #fff;
Aplica uma cor na fonte, nesse caso, branco, definido pelo valor em hexadecimal. Note que o valor em hexadecimal só tem 3 dígitos, isso acontece porque quando cada par de valor hexadecimal se repete, podemos escrever somente um dígito para cada par, por exemplo, #33cc66 pode ser escrito #3c6.

width: 500px;
Se você está realmente testando no navegador cada modificação nos arquivos html e css, deve ter reparado que o figcaption ocupa toda a largura da página e como definimos uma cor de fundo para o figcaption, ficou uma faixa cinza em toda a largura da página, para colocar essa faixa do tamanho da foto, usamos a propriedade width (largura) com o valor de 500 pixels, que é a largura do foto, podemos ver essa largura com o Inspetor de Elementos do Google Chrome.

height: 30px;
Aplica uma altura de 30 pixels.

margin: 0 auto;
Centraliza o figcaption, na verdade esse é um macete do css, pois o que estamos fazendo realmente é definindo uma margem 0 para cima (top) e para baixo (bottom) e uma margem auto (dividir o espaço) para a direita (right) e para a esquerda (left).

padding-top: 6px;
Note que o texto do figcaption ficou muito colado na parte de cima, uma das soluções possíveis para resolver esse problema é aplicar um espaçamento interno só no topo.

box-sizing: border-box;
Se você reparar a propriedade padding se soma ao tamanho do figcaption, os 30 pixels de altura do figcaption se somou aos 6 pixels do padding-top, mas como não queremos que isso aconteça, podemos definir o tamanho da caixa pela borda da caixa, o padding é aplicado sem aumentar o tamanho do figcapiton.

No css não existe só uma maneira correta de fazer um estilo, mais a frente, veremos como fazer esse mesmo layout no header com outras propriedades css.

Código css:

.cabecalho {
    width: 100%;
    height: 80px;
    background-color: #fdd73d;
    font-size: 24px;
}

.corpo {
    width: 100%;
    height: 100%;
    background-color: #f5f5f5;
}

.rodape {
    width: 100%;
    height: 60px;
    background-color: #fdd73d;
    font-size: 24px;
}

.corpo h1 {
    margin-top: 0;
}

.cabecalho p {
    margin-top: 0;
    margin-bottom: 0;
}

.rodape p {
    margin-top: 0;
}

.logo {
    display: inline-block;
    padding: 10px;
}

.nav {
    display: inline-block;
    float: right;
    padding: 10px;
    position: relative;
    top: 50%;
    transform: translateY(-50%);
}

h1, h2, figure {
    text-align: center;
}

figcaption {
    font-family: "Arial", sans-serif;
    font-weight: bold;
    font-style: italic;
    background-color: #686868;
    color: #fff;
    width: 500px;
    height: 30px;
    margin: 0 auto;
    padding-top: 6px;
    box-sizing: border-box;
}
