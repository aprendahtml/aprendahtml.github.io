Propriedade Flex do CSS

Para definir o layout do header usamos no arquivo css, as propriedade display com o valor inline-block e a propriedade float com o valor right, agora vamos tentar fazer o mesmo layout usando a propriedade flex.

É muito mais simples e fácil definir o layout com a propriedade flex, no arquivo css adicione a propriedade display com o valor flex e a propriedade justify-content com o valor space-between, simples assim.

Código html:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <link rel="stylesheet" href="exemplo-flex.css">
    <meta charset="utf-8">
    <title>Exemplo Flex</title>
</head>
<body>
    <header class="cabecalho">
        <div class="logo">
            <img src="img/logo.png" alt="Logo do Site" title="Logo do Site">
        </div>
        <nav class="nav">
            <ul class="menu">
                <li><a href="index.html">Home</a></li>
                <li><a href="fotos.html">Fotos</a></li>
                <li><a href="videos.html">Vídeos</a></li>
                <li><a href="contato.html">Contato</a></li>
            </ul>
        </nav>
    </header>
</body>
</html>

Código css:

.cabecalho {
    display: flex;
    justify-content: space-between;
    width: 100%;
    height: 80px;
    background-color: #fdd73d;
    font-size: 24px;
}

.logo {
    padding: 10px;
}

.menu {
    list-style: none;
    margin: 0;
    padding-top: 25px;
}

.menu li {
    font-family: "Arial", sans-serif;
    font-weight: bold;
    text-transform: uppercase;
    font-size: 20px;
    margin-right: 20px;
    display: inline-block;
}

.menu li a {
    text-decoration: none;
    color: #000;
}

.menu li a:hover {
    color: #686868;
}

Vamos ver outros exemplos com a propriedade flex, para isso vamos criar uma div com o class container e dentro dela vamos criar 3 outras divs com o class box-1, box-2 e box-3, vamos definir no arquivo css, altura, largura e cor de fundo, com as propriedades width, height e background-color e no class container vamos adicionar a propriedade display com o valor flex.

Código html:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <link rel="stylesheet" href="exemplo-flex-1.css">
    <meta charset="utf-8">
    <title>Exemplo Flex 1</title>
</head>
<body>
    <div class="container">
        <div class="box-1">Box 1</div>
        <div class="box-2">Box 2</div>
        <div class="box-3">Box 3</div>
    </div>
</body>
</html>

Código css:

.container {
    display: flex;
    width: 100%;
    height: 200px;
    background-color: yellow;
}

.box-1 {
    width: 200px;
    height: 150px;
    background-color: green;
}

.box-2 {
    width: 200px;
    height: 150px;
    background-color: blue;
}

.box-3 {
    width: 200px;
    height: 150px;
    background-color: magenta;
}

Resultado no navegador:

A propriedade flex-direction, que define a direção de fluxo dos box no container, pode receber os possíveis valores:
row: da esquerda para a direita, é o valor padrão, se a propriedade não for definida esse valor é usado automaticamente.
row-reverse: da direita para a esquerda.
column: de cima para baixo.
column-reverse: de baixo para cima.

Vamos ver o exemplo de row-reverse, faça o teste com os outros valores.

Código css:

.container {
    display: flex;
    flex-direction: row-reverse;
    width: 100%;
    height: 200px;
    background-color: yellow;
}

.box-1 {
    width: 200px;
    height: 150px;
    background-color: green;
}

.box-2 {
    width: 200px;
    height: 150px;
    background-color: blue;
}

.box-3 {
    width: 200px;
    height: 150px;
    background-color: magenta;
}

Resultado no navegador:

A propriedade justify-content, que define o alinhamento horizontal dos box no container, pode receber os possíveis valores:
flex-start: alinha os box à esquerda do container, é o valor padrão, se a propriedade não for definida esse valor é usado automaticamente.
flex-end: alinha os box à direita do container.
center: alinha os box ao centro do container.
space-between: alinha o primeiro box à esquerda, o último box à direita e os outros box são centralizados utilizando o espaçamento disponível.
space-around: é bem parecido com o space-between, a diferença é que é utilizado a metade dos espaços entre os box, antes do primeiro box e depois do último box, os outros box são centralizados utilizando o espaçamento disponível.
initial: usa o valor inicial, que é flex-start.
inherit: usa o valor da tag pai, se o valor não estiver definido, usa o valor padrão, que é flex-start.

Vamos ver o exemplo de space-around, faça o teste com os outros valores.

Código css:

.container {
    display: flex;
    justify-content: space-around;
    width: 100%;
    height: 200px;
    background-color: yellow;
}

.box-1 {
    width: 200px;
    height: 150px;
    background-color: green;
}

.box-2 {
    width: 200px;
    height: 150px;
    background-color: blue;
}

.box-3 {
    width: 200px;
    height: 150px;
    background-color: magenta;
}

Resultado no navegador:

A propriedade flex, define o tamanho do box em relação ao espaço do container, portanto ela não deve ser definida no container e sim em cada box, no exemplo abaixo, temos 3 box dentro do container, no primeiro box vamos dar ao flex o valor 1, no segundo box vamos dar ao flex o valor 3 e no terceiro box vamos dar ao flex o valor 2. O navegador vai somar os valores do flex de todos os box, nesse exemplo é 6, então irá separar o espaço da largura do container por 6 e aplicará 1 espaço para o box-1, 3 espaços para o box-2 e 2 espaços para o box-3.

Vamos ver o exemplo de flex, faça o teste com outros valores.

Código css:

.container {
    display: flex;
    width: 100%;
    height: 200px;
    background-color: yellow;
}

.box-1 {
    flex: 1;
    width: 200px;
    height: 150px;
    background-color: green;
}

.box-2 {
    flex: 3;
    width: 200px;
    height: 150px;
    background-color: blue;
}

.box-3 {
    flex: 2;
    width: 200px;
    height: 150px;
    background-color: magenta;
}
