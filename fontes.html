Fontes

Fonte é o conjunto de caracteres que formam um tipo de letra caracterizado por uma determinada forma. No arquivo html só escrevemos o texto, a escolha do tipo da fonte e toda a sua formatação, como tamanho, cor, alinhamento e outros atributos são especificados no arquivo css.

No arquivo css primeiro definimos o charset com a codificação de caracteres: 

@charset "utf-8";

Propriedade font-family

Para definirmos o tipo da fonte usamos no seletor a propriedade font-family, no valor usamos o nome da fonte: 

h1 {
    font-family: Arial;
}

O navegador só vai usar a fonte Arial se ela estiver instalada no computador, ou dispositivo, do usuário, senão o navegador vai usar uma fonte genérica. Para amenizar esse problema, podemos definir uma ou mais fontes para serem usadas, no caso da primeira não estiver instalada, e no caso de nenhuma delas estiver instalada, podemos definir uma família genérica da fonte.

Existem 5 famílias genéricas de fonte:

serif: são fontes normais com serifa, serifa é um pequeno prolongamento no final das hastes das letras, normalmente são usadas em parágrafos.

sans-serif: são fontes normais sem serifa, normalmente são usadas em títulos.

monospace: são fontes mono-espaçadas, ou seja, todo caracter ocupa a mesma largura, normalmente a letra o ocupa uma largura maior que a letra i, mas em fontes monospace todas as letras têm a mesma largura. As fontes monospace eram usadas em máquinas de escrever e normalmente são usadas em código de programação.

cursive: são fontes cursivas, ou seja, são fontes que têm o estilo de escrita manual.

fantasy: são fontes decorativas, diferentes, elaboradas, de difícil leitura, normalmente usadas em marcas, logotipos, slogan ou títulos. 

h1 {
    font-family: Arial, Helvetica, sans-serif;
}

No exemplo do arquivo css acima, o navegador vai renderizar todos os h1 com a fonte Arial, se o usuário não tiver a fonte Arial instalada, o navegador vai renderizar todos os h1 com a fonte Helvetica, se o usuário também não tiver a fonte Helvetica instalada, o navegador vai renderizar todos os h1 com uma fonte genérica sem serifa.

Exemplo de fontes:

@font-face

Se quisermos que o navegador renderize exatamente a fonte especificada, mesmo que o usuário não tenha a fonte instalada, podemos usar o @font-face onde adicionamos o atributo font-family com o valor do nome que queremos usar para a fonte, não é obrigatório usar o nome da fonte, podemos usar qualquer nome mas devemos utilizar esse mesmo nome no arquivo css, adicionamos também o atributo src com o caminho para o arquivo da fonte, primeiro passamos o caminho local, que será usado caso o usuário tenha a fonte instalada no computador, e depois o caminho url, para que o navegador baixe o arquivo de fonte. 

@font-face {
    font-family: 'Sangue';
    src: local("Nosifer"),
         url("fonts/Nosifer-Regular.ttf");
}

Código html:

<!DOCTYPE html>
<html lang="pt-br">
    <head>
        <link rel="stylesheet" href="exemplo-fontes.css">
        <meta charset="utf-8">
        <title>Exemplo Fontes</title>
    </head>
    <body>

        <p>Testando @font-face com a fonte Nosifer utilizando o nome Sangue.</p>

    </body>
</html>

Código css:

@charset "utf-8";

@font-face {
    font-family: 'Sangue';
    src: local("Nosifer"),
    url("fonts/Nosifer-Regular.ttf");
}

p {
    font-family: Sangue, fantasy;
    font-size: 18px;
    color: #f00;
}

Web Fonts

Também podemos utilizar fontes diretamente da Internet, de repositórios de fontes, vamos utilizar nesse exemplo o Google Fonts.

Entre no endereço https://fonts.google.com, navegue no site e escolha a sua fonte, nesse exemplo eu usei a fonte 'Shadows Into Light' para ser usada diretamente da internet, clique no sinal de +, na parte inferior aparece um banner escrito “1 Family Selected”, clique nesse banner, para utilizar essa fonte copie o código que aparece embaixo de standard dentro de embed e cole dentro da tag head do seu arquivo html, nesse exemplo o código é: 

<link href="https://fonts.googleapis.com/css?family=Shadows+Into+Light" rel="stylesheet">

Copie também o código abaixo de Specify in CSS e cole no arquivo css, nesse exemplo o código é: 

font-family: 'Shadows Into Light', cursive;

Se não for usar a fonte agora, você pode comentar a linha utilizando /* no começo e */ no final.

É possível fazer a referência a mais de uma fonte dentro da tag link, é só adicionar o caracter pipe (|), que é a barra vertical, após o nome da primeira fonte, antes do fechamento das aspas duplas (“) e incluir o nome da outra fonte, nesse exemplo eu vou incluir a fonte Ubuntu, nesse exemplo o código é: 

<link href="https://fonts.googleapis.com/css?family=Shadows+Into+Light|Ubuntu" rel="stylesheet">

Código html:

<!DOCTYPE html>
<html lang="pt-br">
    <head>
        <link rel="stylesheet" href="exemplo-fontes-1.css">
        <link href="https://fonts.googleapis.com/css?family=Shadows+Into+Light|Ubuntu" rel="stylesheet">
        <meta charset="utf-8">
        <title>Exemplo Fontes 1</title>
    </head>
    <body>

        <p class="fonte-shadows">Testando fonte externa "Shadows Into Light".</p>
        <p class="fonte-ubuntu">Testando fonte externa "Ubuntu".</p>

    </body>
</html>

Código css:

@charset "utf-8";

.fonte-shadows {
    font-family: "Shadows Into Light", cursive;
    font-size: 24px;
}

.fonte-ubuntu {
    font-family: Ubuntu, sans-serif;
    font-size: 24px;
}

Propriedade color

Para definir as cores das fontes usamos no arquivo css a propriedade color com o valor da cor a ser utilizada, o valor da cor pode ser passado pelo nome, por hexadecimal, por rgb, por rgba, por hsl ou por hsla, é recomendado utilizar o valor em hexadecimal, no caso de se utilizar o canal alfa (canal de transparência) na cor, é recomendado utilizar o valor em rgba.

Código html:

<!DOCTYPE html>
<html lang="pt-br">
    <head>
        <link rel="stylesheet" href="exemplo-fontes-2.css">
        <meta charset="utf-8">
        <title>Exemplo Fontes 2</title>
    </head>
    <body>

        <p class="color-nome">Testando propriedade color com valor nome</p>
        <p class="color-hexadecimal">Testando propriedade color com valor hexadecimal</p>
        <p class="color-rgb">Testando propriedade color com valor rgb</p>
        <p class="color-rgba">Testando propriedade color com valor rgba</p>
        <p class="color-hsl">Testando propriedade color com valor hsl</p>
        <p class="color-hsla">Testando propriedade color com valor hsla</p>

    </body>
</html>

Código css:

@charset "utf-8";

p {
    font-family: Arial, sans-serif;
    font-size: 24px;
    font-weight: bold;
}

.color-nome {
    color: red;
}

.color-hexadecimal {
    color: #ff0000;
}

.color-rgb {
    color: rgb(255, 0, 0);
}

.color-rgba {
    color: rgba(255, 0, 0, .5);
}

.color-hsl {
    color: hsl(0, 100%, 50%);
}

.color-hsla {
    color: hsla(0, 100%, 50%, .5);
}

Note que mesmo utilizando notações diferentes no valor das cores, todas elas utilizam a cor vermelha, mesmo o rgba e o hsla, que parece que está utilizando uma cor rosa mas está utilizando a cor vermelha com o canal alfa de transparência em 50%.

As cores na Internet utiliza o padrão de cores do monitor, que é rgb, que é a sigla para red, green e blue, respectivamente vermelho, verde e azul. O padrão rgb utiliza 8 bits para cada cor, na base binária, 28 totaliza 256 valores possíveis, de 0 a 255. Com 256 valores disponíveis para vermelho, verde e azul, obtemos 2563 totalizando 16.777.216 cores.

No Google Chrome existe uma extensão, chamada ColorZilla, que ao se colocar o cursor sobre uma cor nos é mostrado o seu valor em rgb e em hexadecimal e clicando na cor podemos ver o seu valor em hsl.

Resultado no navegador:

Note que o valor em hexadecimal é antecedido pelo caracter cerquilha (#) e esse valor é formado por 3 pares de caracteres, cada par é equivalente ao valor de red, green e blue, respectivamente, então em #ffd700, o ff é o valor do vermelho, o d7 é o valor do verde e o 00 é o valor do azul, note também que se cada um dos 3 valores forem de pares iguais, podemos escrever o valor em hexadecimal de forma abreviada, por exemplo, o valor #ffff00 pode ser escrito #ff0.

O valor em rgb também pode ser escrito de forma percentual, por exemplo:

rgb(100%, 0%, 100%)

Essa forma em percentual é equivalente a rgb(255, 0, 255).

O valor em hsl, que é a abreviação para hue (tom), saturation (saturação) e lightness (luminosidade), recebe o valor de hue em graus e os valores de saturation e lightness em percentual, mesmo se o valor do saturation e do lightness forem 0, deve-se colocar o sinal de por cento (%), por exemplo:

hsl(120, 0%, 25%)

Os valores em rgba e hsla seguem os mesmos princípios do rgb e do hsl, a única diferença é a adição do canal alfa de transparência, que recebe um valor de 0 a 1, divididos em decimais, como 0, 0.1, 0.2, ..., 0.9 e 1, podemos suprimir o 0 à esquerda do ponto (.), então 0.1 pode ser escrito de forma abreviada como .1, o valor 0 é totalmente transparente e o valor 1 é totalmente opaco, então quanto menor o valor de alfa a transparência é maior e quanto maior o valor de alfa a transparência é menor.

Propriedade font-size

A propriedade font-size define o tamanho da fonte, aparentemente é bem simples de se usar, é só definir o seletor e aplicar a propriedade font-size com o valor do tamanho da fonte que queremos, é esse valor que começa a complicar, porque temos vários tipos de valores que podemos usar no font-size, por exemplo: 

font-size: xx-small;
font-size: x-small;
font-size: small;
font-size: medium;
font-size: large;
font-size: x-large;
font-size: xx-large;
font-size: smaller;
font-size: larger;
font-size: 14px;
font-size: 1.5em;
font-size: 1.5rem;
font-size: 50%;
font-size: 8ch;
font-size: 2vw;
font-size: 1.5vh;
font-size: 2.2vmin;
font-size: 1.8vmax;
font-size: 3ex;
font-size: 28pt;
font-size: 10in;
font-size: 10cm;
font-size: 100mm;
font-size: 18pc;

Os valores xx-small, x-small, small, medium, large, x-large e xx-large são equivalentes a extra extra pequeno, extra pequeno, pequeno, médio, grande, extra grande e extra extra grande, vamos ver um exemplo desses valores aplicados ao elemento h1:

Resultado no navegador:

O valor em px (pixel) é equivalente ao menor ponto visível na tela, provavelmente seja o valor para tamanho mais utilizado na Internet, um font-size de 14px quer dizer que a fonte terá 14 pixels de altura e o tamanho desse pixel vai variar conforme o dispositivo e a tela em que será exibido. Veja um exemplo de um zoom da letra a com 14 pixels de altura:

O valor em em é equivalente a 100% do tamanho da sua tag pai, por exemplo, se na tag body for atribuído um font-size de valor 14px, e nas tags filhas de body for atribuído um font-size de 1.5em, esse valor é equivalente a 21px, ou seja 14 X 1.5.

Vamos fazer um exemplo com 3 divs, uma dentro da outra, onde o pai da primeira div é a tag body, o pai da segunda div é a tag da div1 e o pai da terceira div é a tag da div2. Em cada div vamos adicionar a propriedade class com o valor em.

No arquivo css vamos adicionar no seletor body a propriedade font-size com o valor 14 pixels. No seletor .em vamos adicionar a propriedade font-size com o valor 1.5em.

Na primeira div o valor do font-size vai ser equivalente a 21px, que é 1.5 X 14px, pois o elemento pai da primeira div, que é a tag body, tem o valor do font-size de 14px.

Na segunda div o valor do font-size vai ser equivalente a 31.5px, que é 1.5 X 21px, pois o elemento pai da segunda div, que é a tag da primeira div, tem o valor do font-size de 21px.

Na terceira div o valor do font-size vai ser equivalente a 47.25px, que é 1.5 X 31.5px, pois o elemento pai da terceira div, que é a tag da segunda div, tem o valor do font-size de 31.5px.

O valor em rem é equivalente a 100% do tamanho da tag root (raiz), ou seja, da tag html.

Vamos fazer o mesmo exemplo acima mas com o valor da class como rem e no arquivo css vamos adicionar no seletor html a propriedade font-size com o valor 18px e vamos adicionar no seletor .rem a propriedade font-size com o valor 1.5rem.

O valor em % (por cento) é relativo ao valor do font-size do seu elemento pai, por exemplo, numa div com font-size de 16px e dentro dela tivermos um parágrafo com o valor do font-size de 150%, esse valor é equivalente a 24px.

O valor de 1ch é equivalente a largura do caracter 0 (zero) da fonte escolhida, portanto esse valor é variável conforme a fonte. Normalmente é utilizada para indentação de parágrafos.

Os valores vw, vh, vmin e vmax são relativos ao valor do viewport. Viewport é a parte visível da página html na tela do dispositivo, que pode ser um monitor, a tela de um notebook, um smartphone, sem contar com a parte não visível se rolarmos a barra de rolagem, portanto se redimensionarmos a janela do navegador diminuindo o seu tamanho, o tamanho do viewport será menor.

O valor vw (viewport width) é equivalente a 1/100 (um centésimo) da largura do viewport, por exemplo, se a largura do viewport for 1200px, 1vw é equivalente a 12px.

O valor vh (viewport height) é equivalente a 1/100 (um centésimo) da altura do viewport, por exemplo, se a altura do viewport for 960px, 1vh é equivalente a 9.6px.

O valor vmin e vmax são equivalentes a 1/100 (um centésimo) da altura ou da largura do viewport, o vmin usa o menor valor entre o width e o height do viewport e o vmax usa o maior valor entre o width e height do viewport, por exemplo, se a largura do viewport for 1200px e a altura do viewport for 800px, 1vmin é equivalente a 8px e 1vmax é equivalente a 12px, e se a largura do viewport for 720px e a altura do viewport for 968px, 1vmin é equivalente a 7.2px e 1vmax é equivalente a 9.68px.

Vamos ver um exemplo com o valor de 10vw, 10vh, 10vmin e 10vmax, redimensione a tela do seu navegador, aumentando e diminuindo a altura e a largura da janela e veja que o tamanho das fontes mudam conforme o tamanho do viewport.

<!DOCTYPE html>
<html lang="pt-br">
    <head>

        <meta charset="utf-8">
        <title>Exemplo Fontes 10</title>
    </head>
    <body>

    <p class="vw">font-size: 10vw</p>
    <p class="vh">font-size: 10vh</p>
    <p class="vmin">font-size: 10vmin</p>
    <p class="vmax">font-size: 10vmax</p>

    </body>
</html>

Código css:

    margin: 0;
}

.vw {
    font-size: 10vw;
}

.vh {
    font-size: 10vh;

    font-size: 10vmin;

Propriedade text-transform

Propriedade text-shadow

A propriedade text-shadow aplica uma sombra ao texto, pode ser passado 4 valores ao text-shadow, o primeiro valor determina o deslocamento horizontal, sendo o valor positivo o deslocamento é para a direita e o valor negativo o deslocamento é para a esquerda, o segundo valor determina o deslocamento vertical, sendo o valor positivo o deslocamento é para baixo e o valor negativo o deslocamento é para cima, o terceiro valor determina o blur radius, que é o raio de desfoque da sombra, esse valor é opcional e se não for usado a sombra será opaca, o quarto valor determina a cor da sombra, esse valor é opcional e se não for usado a sombra terá a mesma cor da fonte. Propriedade white-space

A propriedade white-space aplica um valor aos espaços em branco e as tabulações do texto. O white-space pode receber os seguintes valores com as seguintes aplicações:

white-space: normal;: mais de um espaço em branco, tabulação e quebra de linha serão ignorados pelo navegador, o texto será quebrado ao atingir o limite do container e o restante do texto será exibido na linha de baixo.

white-space: nowrap;: mais de um espaço em branco, tabulação e quebra de linha serão ignorados pelo navegador, o texto não será quebrado ao atingir o limite do container e será exibido todo em uma linha.

white-space: pre;: mais de um espaço em branco, tabulação e quebra de linha serão exibidos pelo navegador, o texto não será quebrado ao atingir o limite do container e será exibido todo em uma linha.

white-space: pre-wrap;: mais de um espaço em branco, tabulação e quebra de linha serão exibidos pelo navegador, o texto será quebrado ao atingir o limite do container e o restante do texto será exibido na linha de baixo.

white-space: pre-line;: mais de um espaço em branco e tabulação serão ignorados pelo navegador, quebras de linha serão exibidos pelo navegador, o texto será quebrado ao atingir o limite do container e o restante do texto será exibido na linha de baixo.

Vamos ver um exemplo com os 5 valores do white-space em um container pequeno para entendermos melhor:

Código html:

<html lang="pt-br">
    <head>
        <link rel="stylesheet" href="exemplo-fontes-22.css">
        <meta charset="utf-8">
        <title>Exemplo Fontes 22</title>
    </head>
    <body>

        <div class="ws-1">
            <p>
                    white-space: normal;
                Com o valor      normal, os múltiplos espaços em branco, as tabulações e as quebras de linha serão ignorados pelo navegador e quando o texto chegar ao limite do container ele será quebrado e continuará na linha de baixo.
            </p>
        </div>

Propriedade word-wrap

A propriedade word-wrap aplica uma quebra de linha no meio da palavra quando essa palavra for maior que o container, o word-wrap pode receber 2 valores: normal, que não faz a quebra de linha no meio da palavra, nesse caso a palavra se for maior que o container, ela irá ultrapassar os limites do container, e break-word que faz a quebra de linha no meio da palavra na posição onde a letra chegar no limite do container continuando a escrever a palavra na linha de baixo.

Código html:

<!DOCTYPE html>
<html lang="pt-br">
    <head>
        <link rel="stylesheet" href="exemplo-fontes-23.css">
        <meta charset="utf-8">
        <title>Exemplo Fontes 23</title>
    </head>
    <body>

        <div>
            <h3>word-wrap: normal;</h3>
            <p class="normal">
                Testando o word-wrap: normal; com palavras superhipermegagigagrandes como inconstitucionalissimamente e endereços da internet como https://aprendahtml.github.com.    
            </p>
        </div>

        <div>
            <h3>word-wrap: break-word;</h3>
            <p class="break-word">
                Testando o word-wrap: break-word; com palavras superhipermegagigagrandes como inconstitucionalissimamente e endereços da internet como https://aprendahtml.github.com. 
            </p>
        </div>

    </body>
</html>

&nbsp; e &shy;

&nbsp; (non-breaking space) e &shy; (soft hyphen) são entidades html (html entities), o &nbsp; aplica um espaço não quebrável, ou seja, as palavras entre esse espaço sempre estarão juntas mesmo se a página for redimensionada e o &shy; se for necessário aplica uma quebra de linha com um hífen no meio da palavra onde ele foi inserido. Tags <br> <wbr> <sub> <sup> <em> <b>

A tag <br> só é aberta e aplica uma quebra de linha.

A tag <wbr> só é aberta e aplica uma quebra de linha no meio de uma palavra se for necessário.

A tag <sub> é aberta e fechada e aplica um subscrito.

A tag <sup> é aberta e fechada e aplica um sobrescrito.

A tag <em> é aberta e fechada e aplica uma ênfase, um itálico.

A tag <br> é aberta e fechada e aplica um negrito.
