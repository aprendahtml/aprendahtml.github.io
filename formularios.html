Formulários

Muitas páginas da Internet usam formulários, seja para fazer login, enviar uma mensagem, fazer contato ou até mesmo no e-commerce. No HTML 5 foi implementado muitos elementos e atributos semânticos para os formulários. Vamos ver alguns exemplos.

Para se implementar um formulário em html, primeiro abrimos e fechamos a tag com o elemento form e dentro dele adicionamos os campos do formulário.

Dentro da tag de abertura form, adicionamos os atributos method, action e name.

No atributo method, nesse exemplo, vamos utilizar o valor post, esse valor poderia ser get, esse atributo define o método de envio do formulário.

No atributo action, nesse exemplo, vamos utilizar o valor enviar.php, esse valor define a ação que vai acontecer quando o formulário for enviado, nesse caso será executado o arquivo enviar.php.

No atributo name, nesse exemplo, vamos utilizar o valor formulario, esse valor define uma variável, a seguir veremos que cada campo do formulário será definida com um name. A tag form sozinha não mostra nada no navegador. 

<body>
    <form method="post" action="enviar.php" name="formulario">

    </form>
</body>

Antes do campo do formulário vamos adicionar um texto para mostrar ao usuário qual informação estamos solicitando, nesse caso o nome do usuário, então vamos inserir o texto Nome:, para isso vamos envolver o texto com a tag label. Se o usuário clicar no texto que está entre a tag label, o cursor vai para o campo de entrada de dados para o usuário digitar, mas é preciso informar ao label para qual campo do formulário ele deve colocar o cursor, para isso usamos o atributo for na tag de abertura label, nesse exemplo inserimos o valor nameId, que será o mesmo valor que vamos utilizar no atributo id, que vamos inserir na tag de abertura do campo do formulário, o que vamos ver mais a frente.

Vamos adicionar no formulário um campo para o usuário digitar o seu nome, para isso utilizamos o elemento input e adicionamos o atributo type, que define o tipo de entrada de dados no formulário, que nesse caso utiliza o valor text, porque o tipo de dado que o usuário vai passar para o formulário para informar o seu nome é um texto.

Vamos adicionar o atributo id com o valor nameId, o mesmo valor que usamos no atributo for na tag label, então estamos ligando o label a este campo do formulário.

Vamos adicionar o atributo name com o valor name, para definir a variável name que irá receber o valor que o usuário digitar, mais a frente veremos como utilizar essa variável.

Vamos adicionar o atributo placeholder com o valor Digite aqui o seu nome, sem o placeholder o campo para digitar o nome fica em branco, com o placeholder esse valor aparece escrito no campo de entrada, como se fosse uma dica para o usuário, que assim que começar a digitar o placeholder é apagado.

Vamos adicionar o atributo required que não recebe valor por ser um atributo booleano, ele informa que esse campo é requerido, é obrigatório, se não for inserido nenhuma informação o formulário não será enviado e será mostrado uma mensagem de erro.

Vamos adicionar o atributo autofocus que não recebe valor por ser um atributo booleando, ele informa que assim que o formulário for lido pelo navegador, o cursor deve ser inserido no campo do formulário com o atributo autofocus, por isso um formulário que tem vários campos só deve ter um campo com autofocus.

Vamos adicionar o atributo minlength com o valor 3, para definir o número mínimo de caracteres que deve ser inserido, nesse caso se o formulário for enviado com o nome com menos de 3 caracteres o formulário não será enviado e será mostrado uma mensagem de erro.

Vamos adicionar o atributo maxlength com o valor 40, para definir o número máximo de caracteres que deve ser inserido, nesse caso o usuário não conseguirá digitar mais de 40 caracteres no campo e mesmo se o usuário copiar um texto com mais de 40 caracteres, quando o texto for colado nesse campo, só vai aparecer 40 caracteres, portanto o maxlength não retorna erro, lembrando que espaço também é contado como caracter.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

        <label for="nameId">Nome:</label>
        <input type="text" id="nameId" name="name" placeholder="Digite aqui o seu nome" required autofocus minlength="3" maxlength="40">

    </form>
</body>

<body>

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

Vamos incluir no formulário um campo para receber a data de nascimento do usuário, a diferença básica é o valor do atributo type que será date, lembrando que os valores dos atributos id e name devem ser únicos para cada campo. Existem outros valores que podemos atribuir ao atributo type para se trabalhar com datas em formulários, como datetime, month, week, time e datetime-local.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

        <label for="dataNascId">Data de Nascimento:</label>
        <input type="date" id="dataNascId" name="datanasc">

    </form>
</body>

Vamos incluir no formulário um campo para receber o endereço do site do usuário, a diferença básica é o valor do atributo type que será url, lembrando que os valores dos atributos id e name devem ser únicos para cada campo e usaremos também o atributo placeholder. Lembrando que no url o usuário precisa digitar http:, https: ou ftp:.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

        <label for="siteId">Endereço do seu site:</label>
        <input type="url" id="siteId" name="site" placeholder="Digite aqui o endereço do seu site">

    </form>

Resultado no navegador:

Vamos incluir no formulário um campo para receber um número do usuário, a diferença básica é o valor do atributo type que será number, lembrando que os valores dos atributos id e name devem ser únicos para cada campo e usaremos também o atributo placeholder, o atributo min, com o valor mínimo que será aceito e o atributo max, com o valor máximo que será aceito. O número pode ser passado pelo usuário digitando diretamente no campo ou clicando nas setas para cima ou para baixo no final do campo.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

        <label for="depId">Dos 513 Deputados Federais, quantos são corruptos?</label>
        <input type="number" id="depId" name="dep" min="0" max="513" placeholder="Digite o número aqui. Não vale escrever todos.">            

    </form>
</body>

Resultado no navegador:

Vamos incluir no formulário um campo para que o usuário escolha um valor através de um controle deslizante utilizando o valor range no atributo type no elemento input, usamos além do id e name, o atributo min, passando o valor mínimo, o atributo max, passando o valor máximo e o atributo list passando o valor do id do elemento datalist que vamos criar para passar os valores do range.

Abaixo da tag input, abrimos e fechamos a tag datalist com o atributo id com o mesmo valor passado ao atributo list do input, nesse exemplo, lista. Dentro da tag datalist, adicionamos o elemento option com o atributo value com cada valor do range, nesse exemplo criamos option de valor 0 até o valor 10.

Entre o label e o input, criamos uma div com um class de valor range e dentro dela adicionamos vários span de 0 a 10 para mostrar esses valores acima do range e para que possamos aplicar os estilos css.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

    <label for="notaId">Nota do Site:</label>
    <div class="range">
        <span>0</span><span>1</span><span>2</span><span>3</span><span>4</span><span>5</span><span>6</span><span>7</span><span>8</span><span>9</span><span>10</span>
    </div>
    <input type="range" id="notaId" name="nota" min="0" max="10" list="lista">
    <datalist id="lista">
        <option value="0">
        <option value="1">
        <option value="2">
        <option value="3">
        <option value="4">
        <option value="5">
        <option value="6">
        <option value="7">
        <option value="8">
        <option value="9">
        <option value="10">
    </datalist>

    </form>
</body>

Resultado no navegador:

Vamos incluir no formulário um campo para o usuário escolher uma cor, a diferença básica é o valor do atributo type que será color, lembrando que os valores dos atributos id e name devem ser únicos para cada campo e usaremos também o atributo value com o valor #ff0000, para mostrar a cor vermelha, por padrão o color mostra a cor preta, vale lembrar que a cor, em value, tem que ser escrito em hexadecimal, com a cerquilha (#) e os seis dígitos, não pode ser escrito de forma resumida, como #f00, essa forma resumida funciona no css.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

    <label for="corId">Escolha uma cor:</label>
    <input type="color" id="corId" name="cor" value="#ff0000">                    

    </form>
</body>

Resultado no navegador:

Vamos incluir no formulário um campo chamado radio button, no qual nós passamos ao usuário algumas opções para ele escolher clicando na opção desejada, normalmente o radio é usado quando o usuário deve escolher somente uma das opções disponíveis. O valor do type é radio e além do id e name, usamos o atributo value com o valor de retorno para cada opção, nesse exemplo para a opção Masculino o valor retornado será masculino e assim por diante, o valor do value é o que vai aparecer para você quando o usuário enviar o formulário, no primeiro radio incluímos o atributo checked, que por ser um atributo booleano, não recebe valor, o checked é para deixar marcado essa opção no formulário, portanto só deve ser usado em uma das opções do radio, note também que envolvemos todo o input e o nome da opção com a tag label, para que o usuário ao clicar no nome da opção, também marque o radio.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

    <label for="sexoId">Qual é o seu sexo?</label><br>
    <label><input type="radio" id="sexoId" name="sexo" value="masculino" checked>Masculino</label>
    <label><input type="radio" id="sexoId" name="sexo" value="feminino">Feminino</label>
    <label><input type="radio" id="sexoId" name="sexo" value="nao informado">Prefiro não informar</label>

    </form>
</body>

Resultado no navegador:

Vamos incluir no formulário um campo chamado caixa de seleção ou combo box, no qual o usuário clica numa caixa como um menu dropbox e escolhe uma opção. Esse campo é diferente dos outros porque não é definido com o elemento input, é definido pelo elemento select, que é aberto e fechado, onde incluímos os atributos id e name.

Dentro do select adicionamos o elemento option, que é aberto e fechado, com o atributo value com o valor vazio, abrindo e fechando aspas duplas ("") e dentro deste primeiro option, com o valor de value vazio, escrevemos uma mensagem para o usuário, como se fosse um placeholder, nesse exemplo escrevemos Escolha aqui o seu time.

Nos próximos option adicionamos o valor em value, conforme o retorno que queremos do formulário, nesse exemplo, na opção Corinthians adicionamos o valor corinthians.

Podemos ainda adicionar o elemento optgroup, como foi feito nesse exemplo, a tag do optgroup só é aberta e dentro dela adicionamos o atributo label com o valor do nome do grupo de opções, nesse exemplo usamos 2 optgroup, um com valor Times Paulistas e outro com valor Times Cariocas, esse valor de label não é uma opção para o usuário escolher, serve também como um placeholder.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

        <br><label for="timeId">Para qual time você torce?</label>
        <select id="timeId" name="time">
            <option value="">Escolha aqui o seu time</option>
        <optgroup label="Times Paulistas">
            <option value="corinthians">Corinthians</option>
            <option value="sao paulo">São Paulo</option>
            <option value="santos">Santos</option>
            <option value="palmeiras">Palmeiras</option>
        <optgroup label="Times Cariocas">
            <option value="flamengo">Flamengo</option>
            <option value="fluminense">Fluminense</option>
            <option value="vasco">Vasco</option>
            <option value="botafogo">Botafogo</option>
        </select>

    </form>
</body>

Resultado no navegador:

Vamos incluir no formulário um campo de texto onde o usuário vai digitar uma resposta, a diferença é que ao clicar no campo de texto aparece algumas opções pré-definidas que o usuário pode escolher e ao digitar alguma letra as opções ficam restritas a essa letra, mas o usuário pode digitar outra resposta qualquer que não esteja listada nas opções.

Esse campo também é criado com o elemento input mas não utiliza o atributo type, utilizamos o atributo list com o valor do id, também não usamos o id no input, usaremos no datalist, adicionamos no input os atributos name e placeholder.

Abrimos e fechamos a tag datalist e na tag de abertura adicionamos o atributo id. Dentro do datalist usamos o elemento option, só com a tag de abertura e com o atributo value com o valor da opção que queremos passar ao usuário. Usamos quantos option que acharmos necessário.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

        <label for="linuxId">Qual Linux você prefere?</label>
        <input list="linuxId" name="linux" placeholder="Responda aqui">
        <datalist id="linuxId">
            <option value="Red Hat">
            <option value="Debian">
            <option value="Ubuntu">
            <option value="Mint">
            <option value="Fedora">
            <option value="CentOs">
            <option value="Suse">
        </datalist>

    </form>
</body>

Resultado no navegador:

Vamos incluir no formulário um campo chamado checkbox, onde nós passamos algumas opções para o usuário escolher, clicando e marcando a opção desejada, no checkbox o usuário pode escolher mais de uma opção.

O checkbox é envolvido por uma caixa ou um quadro, onde é feita uma pergunta ao usuário para que ele possa escolher as opções. Essa caixa é definida pela tag fieldset, que é aberta e fechada, e dentro do fieldset adicionamos a tag legend, que é aberta e fechada, e dentro da tag legend escrevemos o texto com a pergunta ao usuário.

Abaixo da tag legend, ainda dentro do fieldset, incluímos as tags input, que só é aberta, e a tag label, que é aberta e fechada.

Na tag input adicionamos o atributo type com o valor checkbox, adicionamos também os atributos id, name e value. Abaixo da tag input adicionamos a tag label com o atributo for com o valor passado ao id do input, dentro da tag label escrevemos o nome da opção. Após a tag de fechamento label, adicionamos o elemento br para quebrar uma linha, o que não precisa ser adicionado ao último label do fieldset, com a última opção.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

        <fieldset>
            <legend>Qual é o seu estilo musical preferido?</legend>

            <input type="checkbox" id="mpbId" name="musica" value="mpb">
            <label for="mpbId">MPB</label><br>

            <input type="checkbox" id="rockId" name="musica" value="rock">
            <label for="rockId">Rock</label><br>

            <input type="checkbox" id="popId" name="musica" value="pop">
            <label for="popId">Pop</label><br>

            <input type="checkbox" id="sertanejoId" name="musica" value="sertanejo">
            <label for="sertanejoId">Sertanejo</label><br>

            <input type="checkbox" id="pagodeId" name="musica" value="pagode">
            <label for="pagodeId">Pagode</label><br>

            <input type="checkbox" id="funkId" name="musica" value="funk">
            <label for="funkId">Funk</label>
        </fieldset>

    </form>
</body>

Resultado no navegador:

Vamos incluir no formulário um campo chamado textarea que é uma área para o usuário incluir um texto, normalmente usado para o usuário enviar uma mensagem.

Para adicionar o textarea não usamos a tag input, usamos a tag textarea, que é aberta e fechada, dentro da tag de abertura do textarea adicionamos os atributos id, name e placeholder. Podemos, ainda, incluir o atributo maxlength com o valor em número, nesse exemplo demos o valor de 400, esse valor define o número máximo de caracteres que o usuário pode digitar no campo de texto, lembrando que espaço também é contado como caracter.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

        <label for="messageId">Mensagem:</label>
        <textarea id="messageId" name="message" placeholder="Digite aqui a sua mensagem" maxlength="400"></textarea>

    </form>
</body>

Resultado no navegador:

Para finalizar o nosso formulário vamos incluir um botão para que o usuário possa enviar o formulário. Para isso usamos a tag input com o atributo type com o valor submit, o atributo name com o valor submit e com o atributo value com o valor que você queira escrever no botão, nesse exemplo demos o valor Enviar.

Código html:

<body>
    <form method="post" action="enviar.php" name="formulario">

        <input type="submit" name="submit" value="Enviar">

    </form>
</body>
