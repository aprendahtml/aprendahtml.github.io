Áudio e Vídeo

Para adicionar um áudio na página html, abrimos e fechamos a tag audio, dentro da tag audio adicionamos o elemento html source, que só tem tag de abertura, e adicionamos o atributo src, com o caminho para o arquivo de áudio, no exemplo abaixo o valor é media/whatsapp.mp3 e o atributo type, com o valor do tipo do arquivo, no exemplo abaixo o valor é audio/mpeg. Os elementos escritos dentro da tag audio não serão exibidos pelo navegador, por exemplo um parágrafo, um título ou uma imagem, se quiser inserir algum elemento, o faça antes ou depois da tag audio. Para mostrar os controles de Play, Tempo e Volume, adicione o atributo controls na tag de abertura do elemento audio, esse atributo é booleano, portanto não precisa de valor.

Adicionamos um título para o áudio com a tag h2, um link para download do áudio com a tag a e uma imagem de download com a tag img. Na tag a, além do atributo href, com o caminho para o arquivo de áudio, adicionamos o atributo download, o valor do atributo download é o nome que o usuário vai baixar o arquivo de áudio, no exemplo abaixo, o caminho do arquivo é media/whatsapp.mp3 e o valor do atributo download é toque-download.mp3, nesse caso o usuário vai fazer o download do arquivo whatsapp.mp3 mas o arquivo vai ser baixado com o nome toque-whatsapp.mp3. Para facilitar os estilos, vou englobar todas as tags numa div com class media. Note que a imagem de download ficou ao lado dos controles de áudio, isso acontece porque os elementos audio, img e a são elementos inline, resolvemos esse problema aplicando no seletor audio a propriedade display com o valor block.

Código html:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <link rel="stylesheet" href="exemplo-audio-e-video.css">
    <meta charset="utf-8">
    <title>Exemplo Áudio e Vídeo</title>
</head>
<body>
    <div class="media">
        <h2>Toque do WhatsApp</h2>
        <audio controls>
            <source src="media/whatsapp.mp3" type="audio/mpeg">
        </audio>
        <a href="media/whatsapp.mp3" download="toque-whatsapp.mp3"><img src="img/download.png" alt="Clique para fazer o Download"></a>
    </div>
</body>
</html>

Código css:

.media {
    background-color: #e5e5e5;
    width: 400px;
    margin: 0 auto;
    text-align: center;
    padding-top: 5px;
    border-radius: 5px;
    box-shadow: 2px 2px 2px #000;
}

audio {
    display: block;
    margin: 0 auto;
    padding: 5px 0;
}

.media img {
    width: 200px;
    height: 75px;
    padding: 5px;
}

.media h2 {
    font-family: Arial, sans-serif;
    font-weight: bold;
    font-size: 18px;
}

Para adicionar um vídeo na página html é praticamente o mesmo procedimento para adicionar um áudio, apenas trocamos o elemento audio pelo elemento video e definimos a largura (width) do vídeo no arquivo css, no seletor video. Também trocamos o valor do atributo type do elemento source para video/mp4, se utilizarmos um vídeo em mp4.

Código html:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <link rel="stylesheet" href="exemplo-audio-e-video-1.css">
    <meta charset="utf-8">
    <title>Exemplo Áudio e Vídeo 1</title>
</head>
<body>
    <div class="media">
        <h2>Vídeo de Amostra</h2>
        <video controls>
            <source src="media/sample-720x480.mp4" type="video/mp4">
        </video>
        <a href="media/sample-720x480.mp4" download="sample.mp4"><img src="img/download.png" alt="Clique para fazer o Download"></a>
    </div>
</body>
</html>

Código css:

.media {
    background-color: #e5e5e5;
    width: 500px;
    margin: 0 auto;
    text-align: center;
    padding-top: 5px;
    border-radius: 5px;
    box-shadow: 2px 2px 2px #000;
}

video {
    width: 400px;
    display: block;
    margin: 0 auto;
    padding: 5px 0;
}

.media img {
    width: 200px;
    height: 75px;
    padding: 5px;
}

.media h2 {
    font-family: Arial, sans-serif;
    font-weight: bold;
    font-size: 18px;
}

Na Internet é muito comum inserir vídeos do YouTube nas páginas html, isso é muito fácil de fazer, se chama embed, que é embutir.

No site do YouTube escolha o vídeo que você quer inserir, abaixo do vídeo, clique em Compartilhar, no quadro que abriu, clique em Incorporar, no outro quadro que abriu, escolha as Opções de Incorporação e clique em Copiar, agora é só colar o código na sua página html.

Como trocamos no arquivo html, o elemento audio ou video pelo elemento iframe, o trocamos também no arquivo css e como não podemos fazer download do YouTube, de forma direta, retiramos no arquivo html, as tags a e img e retiramos no arquivo css, o seletor .media img.

Ao retirar o link e a imagem do botão de Download, o vídeo do elemento iframe ficou muito colado com o fundo da div, resolvemos esse problema adicionando no arquivo css, um padding-bottom do seletor iframe, como já temos a propriedade padding com o valor 5px 0, esse valor equivale a 5px para padding-top e padding-bottom e 0px para padding-right e padding-left, como queremos mudar o valor do padding-bottom, usamos os 4 valores na propriedade padding, 5px 0 25px 0, lembrando que o primeiro valor é do top (cima) e vai girando em sentido horário, right (direita), bottom (baixo) e left (esquerda).

Código html:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <link rel="stylesheet" href="exemplo-audio-e-video-2.css">
    <meta charset="utf-8">
    <title>Exemplo Áudio e Vídeo 2</title>
</head>
<body>
    <div class="media">
        <h2>Vídeo do YouTube</h2>
        <iframe width="560" height="315" src="https://www.youtube.com/embed/CH1XGdu-hzQ?rel=0" frameborder="0" allowfullscreen></iframe>
    </div>
</body>
</html>

Código css:

.media {
    background-color: #e5e5e5;
    width: 500px;
    margin: 0 auto;
    text-align: center;
    padding-top: 5px;
    border-radius: 5px;
    box-shadow: 2px 2px 2px #000;
}

iframe {
    width: 400px;
    display: block;
    margin: 0 auto;
    padding: 5px 0 25px 0;
}

.media h2 {
    font-family: Arial, sans-serif;
    font-weight: bold;
    font-size: 18px;
}
