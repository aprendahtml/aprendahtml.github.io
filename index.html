HTML - CSS - JavaScript

Aqui você vai aprender o básico para criar páginas html, utilize o menu à esquerda para navegar entre os tópicos ou os botões de Anterior e Próximo.

Atrelado ao HTML, onde nós inserimos textos, imagens, vídeos e outros objetos à página, existe o CSS que é essencial para a construção de páginas para a Internet, onde nós inserimos tipos de fontes (letras), cores, formatação, alinhamento, posicionamento, bordas, espaçamento e muitos outros estilos à página, por isso também o abordaremos basicamente, e para as animações e interatividade dos elementos nós usamos o JavaScript, que também abordaremos basicamente com alguns exemplos.

Junte-se a nós nesse projeto e participe enviando artigos sobre os assuntos relacionados.
